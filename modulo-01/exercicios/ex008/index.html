Nessa frase temos um termo em negrito usando a tag <strong> (semântica)

Itálico / Ênfase

Nessa frase temos um termo em itálico usando a tag <i> (não semântica)

Nessa frase temos um termo em itálico usando a tag <em> (semântica)

Texto marcado

Também podemos criar um texto marcado usando a tag <mark>

E em outro paragrafo, podemos criar um texto marcado com outra cor usando o parâmetro style.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno.

Texto excluído

Podemos marcar um texto como excluído para indicar que ele deve ser lido pelo usuário, mas não considerado. Usa-se a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Usa-se a tag <ins>

Existe também o texto sublinhado com a tag <u> (não semântica).

Texto sobrescrito

Feito para inserir coisas do tipo x20+3. Usa-se a tag <sup>

Texto subscrito

Feito para inserir coisas do tipo H2O. Usa-se a tag <sub>
